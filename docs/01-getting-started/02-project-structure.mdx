本页面提供了 Next.js 应用的项目结构概况。它包含顶级文件和文件夹、配置文件及`app`和`pages`目录的路由约定。

## 顶级文件夹

使用顶级文件夹来组织您的应用代码和静态文件。

|                                                                                  |                        |
| -------------------------------------------------------------------------------- | ---------------------- |
| [`app`](/docs/content/app/building-your-application/routing)                     | App 路由器             |
| [`pages`](/docs/content/pages/building-your-application/routing)                 | Pages 路由器           |
| [`public`](/docs/content/app/building-your-application/optimizing/static-assets) | 用于存储静态文件       |
| [`src`](/docs/content/app/building-your-application/configuring/src-directory)   | 可选的应用程序源文件夹 |

## 顶级文件

使用顶级文件来配置您的应用、管理依赖、运行中间件、集成监控工具和定义环境变量。

|                                                                                                     |                                |
| --------------------------------------------------------------------------------------------------- | ------------------------------ |
| **Next.js**                                                                                         |                                |
| [`next.config.js`](/docs/content/app/api-reference/next-config-js)                                  | Next.js 的配置文件             |
| [`package.json`](/docs/getting-started/installation#manual-installation)                            | 项目依赖和脚本                 |
| [`instrumentation.ts`](/docs/content/app/building-your-application/optimizing/instrumentation)      | OpenTelemetry and 设备文件     |
| [`middleware.ts`](/docs/content/app/building-your-application/routing/middleware)                   | Next.js 请求中间件             |
| [`.env`](/docs/content/app/building-your-application/configuring/environment-variables)             | 环境变量                       |
| [`.env.local`](/docs/content/app/building-your-application/configuring/environment-variables)       | 本地环境变量                   |
| [`.env.production`](/docs/content/app/building-your-application/configuring/environment-variables)  | 生产环境变量                   |
| [`.env.development`](/docs/content/app/building-your-application/configuring/environment-variables) | 开发环境变量                   |
| [`.eslintrc.json`](/docs/content/app/building-your-application/configuring/eslint)                  | ESLint 的配置文件              |
| `.gitignore`                                                                                        | 要忽略的 Git 文件和文件夹      |
| `next-env.d.ts`                                                                                     | Next.js 的 TypeScript 声明文件 |
| `tsconfig.json`                                                                                     | TypeScript 的配置文件          |
| `jsconfig.json`                                                                                     | JavaScript 的配置文件          |

## `app` 路由约定

The following file conventions are used to define routes and handle metadata in the [`app` router](/docs/content/app).
使用以下文件约定来定义路由和处理 metadata 在[`app`路由器](/docs/content/app)中。

### 路由文件

|                                                                                         |                     |                  |
| --------------------------------------------------------------------------------------- | ------------------- | ---------------- |
| [`layout`](/docs/content/app/api-reference/file-conventions/layout)                     | `.js` `.jsx` `.tsx` | 布局             |
| [`page`](/docs/content/app/api-reference/file-conventions/page)                         | `.js` `.jsx` `.tsx` | 页面             |
| [`loading`](/docs/content/app/api-reference/file-conventions/loading)                   | `.js` `.jsx` `.tsx` | 加载中 UI        |
| [`not-found`](/docs/content/app/api-reference/file-conventions/not-found)               | `.js` `.jsx` `.tsx` | 未找到页面 UI    |
| [`error`](/docs/content/app/api-reference/file-conventions/error)                       | `.js` `.jsx` `.tsx` | 错误 UI          |
| [`global-error`](/docs/content/app/api-reference/file-conventions/error#global-errorjs) | `.js` `.jsx` `.tsx` | 全局错误 UI      |
| [`route`](/docs/content/app/api-reference/file-conventions/route)                       | `.js` `.ts`         | API 断电         |
| [`template`](/docs/content/app/api-reference/file-conventions/template)                 | `.js` `.jsx` `.tsx` | 重新渲染布局     |
| [`default`](/docs/content/app/api-reference/file-conventions/default)                   | `.js` `.jsx` `.tsx` | 平行路由回退页面 |
