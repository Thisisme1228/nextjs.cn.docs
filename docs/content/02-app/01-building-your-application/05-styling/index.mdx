Next.js 支持多种方式来为您的应用程序设置样式，包括：

- **CSS Modules**: 创建局部作用域的 CSS 类，以避免命名冲突并提高可维护性。
- **Global CSS**: 简单易用，对于有传统 CSS 经验的人来说很熟悉，但随着应用程序的增长，可能会导致较大的 CSS 包和难以管理的样式。
- **Tailwind CSS**: 一个实用优先的 CSS 框架，通过组合实用类以允许快速自定义设计。
- **Sass**: 一个流行的 CSS 预处理器，通过添加变量、嵌套规则和 mixins 等功能扩展了 CSS。
- **CSS-in-JS**: 将 CSS 直接嵌入到您的 JavaScript 组件中，实现动态和局部作用域的样式。
