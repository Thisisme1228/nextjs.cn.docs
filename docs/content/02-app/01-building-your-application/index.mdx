Next.js 提供创建灵活的、全栈的 Web 应用的构建块。构建应用程序的指南解释如何使用这些特性及如何自定义应用程序的行为。

这些部分和页面是由基础到高级的顺序组建的，因此您能够遵循它们来逐步构建您的 Next.js 应用程序。不过，您也可以按顺序使用或调到适用您的案例页面。

如果您不熟悉 Next.js,我们推荐您以 [路由](/docs/content/app/building-your-application/routing), [渲染](/docs/content/app/building-your-application/rendering), [数据获取](/docs/content/app/building-your-application/data-fetching) 和 [样式](/docs/content/app/building-your-application/styling) 部分开始使用，它们会以向您介绍基础的 Next.js 和 Web 概念开始。然后，您可以深入了解其他部分，如：[优化](/docs/content/app/building-your-application/optimizing) 和 [配置](/docs/content/app/building-your-application/configuring)。最后，准备好后，请查看[部署](/docs/content/app/building-your-application/deploying) 和 [升级](/docs/content/app/building-your-application/upgrading)部分。
